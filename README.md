##StarWarsMemeSelector

A Star Wars-themed meme generator built with HTML, CSS, and JavaScript. Users can browse characters from the Star Wars universe, filter animated GIFs, and generate random memes through an interactive interface. This project was created as part of my web development journey to strengthen my JavaScript skills while building a fun and engaging application.

##Features
Browse Star Wars characters using dynamically generated radio buttons
Filter results to display only animated GIFs
Random meme selection based on the chosen character
Interactive modal for displaying memes
Custom Star Wars-inspired styling and theme
Responsive and user-friendly interface

##How It Works
Select a Star Wars character from the list.
Optionally enable the Animated GIFs Only filter.
Click the Get Meme button.
A random meme matching the selected character is displayed in a modal window.
Close the modal and generate another meme whenever you like.
